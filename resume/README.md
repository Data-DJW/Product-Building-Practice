# Modern Resume Website Template

A modern, responsive resume website template built with a Cobalt Blue and Black theme.

## Features
- **Modern Design**: Professional color palette and typography.
- **Web Components**: Built using native Web Components for modularity.
- **Responsive**: Works on mobile and desktop devices.
- **Sections**:
  - **Bio**: Professional summary and introduction.
  - **Career**: Experience timeline.
  - **Inquiry**: Contact form for potential collaborations.

## Technologies
- HTML5
- CSS3 (Vanilla CSS with Flexbox/Grid)
- JavaScript (Vanilla ES Modules & Web Components)
- Font Awesome (Icons)
- Google Fonts (Inter)
