# Project Blueprint: Modern Tetris

## Overview
A modern, responsive Tetris game built with HTML5 Canvas, CSS, and vanilla JavaScript. The application features a clean, high-contrast UI with support for both Dark and Light modes.

## Current State & Features
- **Core Gameplay**: Tetromino movement, rotation, line clearing, and level-based scoring.
- **Visuals**: Neon-colored blocks on a dark background (Dark Mode) or vibrant colors on a clean background (Light Mode).
- **Responsiveness**: Layout adapts to various screen sizes.
- **Theme**: User-selectable Dark/Light mode using CSS variables.

## Implementation Plan (Task: Initial Build)
1. **Setup index.html**: Create the game container, canvas, and UI controls.
2. **Setup style.css**: Implement CSS variables for themes and overall layout.
3. **Setup main.js**: Implement the Tetris game engine and theme toggle logic.
4. **Verification**: Confirm gameplay mechanics and theme switching.
