# Project Blueprint: Modern Resume Website

## Overview
A modern, responsive resume website built using HTML5, CSS, and Vanilla JavaScript with custom Web Components. The design features a professional Cobalt Blue and Black color scheme, with a focus on clean typography and interactive elements.

## Design & Visuals
- **Color Palette**: 
  - Background: Black/Deep Gray (#0c0c0c)
  - Primary Accent: Cobalt Blue (#0047AB)
  - Secondary Accent: Light Cobalt (#3373C4)
  - Text: Off-white/Light Gray (#f0f0f0)
- **Typography**: Modern sans-serif fonts (e.g., system fonts or Inter-like fallbacks).
- **Layout**: Modular design using Web Components for sections, optimized for both desktop and mobile.
- **Interactivity**: Smooth transitions, hover effects on buttons, and a clean, responsive contact form.

## Current State & Features
- **Section: Bio**: Introduces the individual with a profile summary, name, and professional title.
- **Section: Career**: Lists professional experiences in a timeline or structured format.
- **Section: Inquiry**: Provides a contact point for visitors, including a mock inquiry form.

## Implementation Steps
1. **HTML Structure (`index.html`)**: Entry point with a main layout container and custom component tags.
2. **Global Styling (`style.css`)**: Defines the color scheme, layout, and common utility classes.
3. **Web Components (`main.js`)**: Defines `<resume-bio>`, `<resume-career>`, and `<resume-inquiry>` elements with encapsulated logic and styling.
4. **Verification**: Confirm visual consistency, responsiveness, and component functionality.
